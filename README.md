# 🎨 AI Virtual Painter

A real-time gesture-controlled virtual drawing application using computer vision.  
Draw in the air using your fingers with the help of a webcam.

---

## 🚀 Features

- ✍️ Draw using index finger
- ✌️ Gesture-based controls
- 🎨 Color selection (Red, Green, Blue)
- 🧼 Eraser tool
- 🧹 Clear screen option
- 💾 Save drawing as image
- 🤖 Real-time hand tracking using MediaPipe

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py



| Gesture / Key   | Action       |
| --------------- | ------------ |
| ☝️ Index Finger | Draw         |
| ✌️ Two Fingers  | Select Tool  |
| ✊ Fist          | Stop Drawing |
| `S` Key         | Save Drawing |
| `C` Key         | Clear Screen |
| `Q` Key         | Quit         |
